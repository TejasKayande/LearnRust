# Rust Book Practice (Cargo-Based)

This repository contains my hands-on practice while learning Rust from
*The Rust Programming Language* (The Rust Book).

Based on the official Rust book:
https://doc.rust-lang.org/book/

## How to Navigate

* Built using **Cargo**
* Each concept/lesson is captured as a **separate commit**

Since each lesson is stored as a commit:

```bash
git log --oneline
```

Browse a specific step:

```bash
git checkout <commit-hash>
```

Return to latest:

```bash
git checkout master
```

## Running the Project

Make sure Rust and Cargo are installed:

```bash
cargo --version
```

Build the project:

```bash
cargo build
```

Run the project:

```bash
cargo run
```
